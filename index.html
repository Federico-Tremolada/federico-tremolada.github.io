About me

Technical positioning for a biomechanical engineering portfolio

Profile

I am an MSc Biomedical Engineering student at Politecnico di Milano, currently focused on biomechanics, finite element modeling, biomaterials, and engineering analysis for medical-device-oriented applications.

My goal is to build practical projects that combine simulation, data processing, and physically meaningful interpretation. I am especially interested in work where mechanics, numerical modeling, and biomedical applications come together in a rigorous but usable way.

Selected projects

Current portfolio projects: one fully developed, others already defined as next steps

enthesis-fem-model
Public

Finite element study of the tendon-to-bone interface, comparing sharp and functionally graded transitions. The project includes geometry definition, material assignment, boundary conditions, S11 extraction, SENER analysis, refinement strategy, exponent sensitivity, and comparison between constant and variable Poisson ratio.

Abaqus Python MATLAB Biomechanics
Python Main portfolio project
sla-fatigue-analysis
In progress

Mechanical characterization workflow for SLA-printed polymer specimens under four-point bending and fatigue loading. The project combines test interpretation, machine data processing, cycle selection, force-displacement analysis, and engineering discussion of material behavior.

Mechanical testing MATLAB Fatigue SLA
MATLAB Next portfolio project

Collection of scripts and workflows for data cleaning, stiffness extraction, signal smoothing, regression, cycle-based comparisons, and engineering plots for biomechanical simulation and experimental datasets.

Development path

How the work is evolving from technical study to public engineering portfolio

Step 01

Model construction

Build simplified geometries, define materials, assign boundary conditions, and compare baseline solutions in a controlled FEM setup.

Step 02

Post-processing pipeline

Extract physically meaningful variables with Python and MATLAB, organize results, and generate plots suitable for technical discussion.

Step 03

Project refinement

Improve model quality through refinement, parameter sensitivity, and advanced comparisons such as graded laws and energy-based indicators.

Step 04

Public output

Transform the work into a GitHub repository, portfolio page, and paper-ready narrative with clear figures, methods, and conclusions.
