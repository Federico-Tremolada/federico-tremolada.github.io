Selected projects

Current portfolio projects: one fully developed, others already defined as next steps

enthesis-fem-model
Public

Finite Element model of the tendon-to-bone interface (enthesis) developed in Abaqus. The project investigates how different material transition laws—sharp, linear, exponential, and power-law—affect stress distribution (S11) and elastic energy transfer (SENER). Post-processing was carried out with Python and MATLAB to extract interface data, compare configurations, and identify the most effective graded solution.

Abaqus Python MATLAB Biomechanics
Python Main portfolio project
sla-fatigue-analysis
In progress

Mechanical characterization workflow for SLA-printed polymer specimens under four-point bending and fatigue loading. The project combines test interpretation, machine data processing, cycle selection, force-displacement analysis, and engineering discussion of material behavior.

Development path

How the work is evolving from technical study to public engineering portfolio

Step 01

Model construction

Build simplified geometries, define materials, assign boundary conditions, and compare baseline solutions in a controlled FEM setup.

Step 02

Post-processing pipeline

Extract physically meaningful variables with Python and MATLAB, organize results, and generate plots suitable for technical discussion.

Step 03

Project refinement

Improve model quality through refinement, parameter sensitivity, and advanced comparisons such as graded laws and energy-based indicators.

Step 04

Public output

Transform the work into a GitHub repository, portfolio page, and paper-ready narrative with clear figures, methods, and conclusions.
